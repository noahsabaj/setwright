# Setwright release lockups

The lockups in this directory reproduce the supplied monochrome identity as
resolution-independent SVG. Keep the wordmark, tagline, description, spacing,
and black-and-white treatment together for splash screens, onboarding, store
art, and release pages.

The application workspace may use the product palette described in
`docs/brand.md`; these release lockups remain monochrome. Convert text to
outlines in final platform-store artwork when identical typography across
renderers is required.
