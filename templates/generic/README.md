# Generic article

A conservative `article` starting point for preprints and research drafts. It
uses widely available TeX Live packages for math, theorem/proof structures,
`booktabs` tables, figures, `listings` code, links, and a BibTeX bibliography.

Compile with `latexmk -pdf -norc main.tex` in a trusted TeX environment, or use
Setwright's managed sandbox once that release gate is complete.
