# ACM `acmart` article

This template uses `acmart`'s `sigconf`, `review`, and `anonymous` options for a
conference review draft. Replace the placeholder conference metadata and
compare every option with the current ACM venue instructions before submission;
camera-ready requirements differ.

Compile with `latexmk -pdf -norc main.tex` in a trusted TeX environment, or use
Setwright's managed sandbox once that release gate is complete.
